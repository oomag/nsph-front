A NEW STEP IN THE EVOLUTION OF CLOUD TECHNOLOGY

The Noosphere Testnet consists of two elements

THE FIRST is a purely virtual network supplied as part of the Shard Designer and emulating the operation of the Noosphere network on a single computer.

THE SECOND is a live test network created and supported by the Noosphere Foundation for full-scale testing of new shards, especially when candidates for static shards appear.

TESTNET
Noosphere is a blockchain-based, service-oriented sharding platform

designed to provide cloud services and broad cloud computing capabilities. The platform incorporates a multi-blockchain cloud ecosystem in which blockchain technology and cloud solutions can be utilized to the fullest advantage by absolutely anyone.

A whole new level of corporate cloud solutions like Azure, AWS or Google Cloud taking scalability and data protection capabilities to new heights.

DAR & DDNS
(Dynamic Application Routing & Dynamic Domain Name System) - a service that ensures load balancing, data stream routing and IP-level fault tolerance with integrated dynamic DNS functionality for any applications.
DDAP
(Decentralized Directory Access Protocol) - a service offering LDAP functionality with enhanced safety and fault tolerance requirements.
LOKI
- a service for protection against DDoS attacks that enables timely detection of threats by way of continuous communication channel monitoring and redirects traffic to special nodes thus simulating success of attacks.
DHPC
(Decentralized High Performance Computing) - a service that facilitates transfer and adaptation of existing computing tasks and software to the Noosphere system for the use of distributed computing capacity and is also involved in this type of computing.
NTM
(Noosphere Transaction Mixer) - an additional transaction animation service usable not only for any cryptocurrency, but also in transmission of any confidential data.
ACS
(Autonomous Copyright System) - a service providing copyright functions.
EBS
(Effective Backup Service) - a fast and efficient data backup service that makes it possible to boost by dozens of times the speed of backup and snapshot generation as compared with centralized services due to automatic data sharing between backup nodes in the course of transmission.
PVM
(Python Virtual Machine) - a virtual smart contract processing machine based on a Python-like language.
ABG
(Any Blockchain Gate) - a template service that provides unified API for operations with any third-party blockchain.
