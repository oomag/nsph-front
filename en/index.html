A NEW STEP IN THE EVOLUTION OF CLOUD TECHNOLOGY

The Noosphere Testnet consists of two elements

THE FIRST is a purely virtual network offered as a part of the Shard Designer and emulating the operation of the Noosphere network on a single computer.

THE SECOND is a live test network created and supported by the Noosphere Foundation for full-scale testing of new shards, especially when candidates for static shards appear.

TESTNET
Noosphere is a blockchain-based, service-oriented sharding platform

designed to provide cloud services and broad cloud computing capabilities. The platform incorporates a multi-blockchain cloud ecosystem in which blockchain technology and cloud solutions can be utilized to the fullest extent by absolutely anyone.

A whole new level of corporate cloud solutions akin to Azure, AWS or Google Cloud taking scalability and data protection capabilities to new heights.

DAR & DDNS
(Dynamic Application Routing & Dynamic Domain Name System) is a service that offers load balancing, data stream routing and IP-level fault tolerance with integrated dynamic DNS functionality for any applications.
DDAP
(Decentralized Directory Access Protocol) is a service offering LDAP functionality with strengthened safety and fault tolerance parameters.
LOKI
is a service offering protection against DDoS attacks that lets you promptly detect cyber threats by continuously monitoring communication channels and redirects traffic to designated nodes thus simulating success of attacks.
DHPC
(Decentralized High Performance Computing) is a service that facilitates transfer and adaptation of existing computing tasks and software to the Noosphere system in order to use distributed computing power.
NTM
(Noosphere Transaction Mixer) is an additional transaction anonymization service applicable not only to cryptocurrencies, but also in transmission of any confidential data.
ACS
(Autonomous Copyright System) is a service providing copyright functions.
EBS
(Effective Backup Service) is a fast and efficient data backup service that can make the backup and snapshot generation speed several times higher compared to centralized services due to automatic data sharing between backup nodes during transmission.
PVM
(Python Virtual Machine) is a virtual smart contract processing machine based on a Python-like language.
ABG
(Any Blockchain Gate) is a template service that offers a unified API for operations with any third-party blockchain.
